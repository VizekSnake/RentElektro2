# RentElektro Frontend

This is the frontend service for the RentElektro project, built with Vue.js.

## Features

- User interface for managing tool rentals
- Integration with the backend FastAPI service
- Responsive design

## Technology Stack

- **Framework:** Vue.js
- **Build Tool:** Vue CLI
- **Containerization:** Docker

## Prerequisites

- Docker and Docker Compose
- Node.js and npm (for local development)
